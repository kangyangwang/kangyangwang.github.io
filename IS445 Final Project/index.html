Data Introduction

The dataset[1] used in this project shows the information on destructive earthquakes from 1960 to the present that meet at least one of the following criteria: Moderate damage (approximately $1 million or more),10 or more deaths, Magnitude 7.5 or greater, Modified Mercalli Intensity X or greater, or the earthquake generated a tsunami.

The orginal data was obtained from the “Natural Hazards” database of NOAA (National Centers For Environmental Information). Specifically, I selected the “Min” Year as “1960″ on the Significant Earthquake Database, and downloaded the output tsv file on Nov 10th 2021.

The users can use the above link I provided to download the lastest version of the significant earthquake dataset, or use the local file I submitted on Moodle submission page. I also uploaded my downloaded version to my github.io IS445 Final Project directory.

Data Preprocessing

The orginal data includes 38 columns and 2476 rows.

  • Use the python pandas to read the tsv data file and seperate the “Location Name” column into two columns “Country” and “Location”.
import pandas as pd
df = pd.read_csv ("earthquakes-2021-11-10_09-10-19_-0600.tsv", sep = '\t')
df = df.drop(columns=['Search Parameters']).drop([0])
df["Year"] = df["Year"].astype(int)
df[["Country","Location"]] = df["Location Name"].str.split(":",1,expand=True)
  • Fill empty cells in “Total Deaths”, “Total Missing”, “Total Injuries” columns with 0 and create a new column “Population” by adding up these columns and dividing by 1000.
df["Total Deaths"].fillna(value=0, inplace=True)
df["Total Missing"].fillna(value=0, inplace=True)
df["Total Injuries"].fillna(value=0, inplace=True)
df["Population"] = (df["Total Deaths"] + df["Total Missing"] + df["Total Injuries"])/1000
  • Only keep the columns that are useful in this project, and then save the processed data into “earthquake_processed.csv” file to be uploaded to my GitHub IS455 Final Project directory.
df = df[["Year", "Country", "Latitude", "Longitude", "Focal Depth (km)", "Mag", "MMI Int", "Total Deaths", 
         "Total Missing", "Total Injuries", "Total Damage ($Mil)", "Total Houses Destroyed", "Total Houses Damaged", "Population"]]
df.to_csv('earthquake_processed.csv', index=True)

4. The Map Distribution of Tsunamis Since 1960 -- 2nd Contextual Dataset: Tsunamis Since 1960

Another contextual dataset used here is Tsunami Data[2] since 1960. This dataset was also downloaded from NOAA and has been preprocessed in the same method as the earthquake dataset and uploaded to my github.io IS445 Final Project folder. With this dataset, we can compare the tsunamis with the earthquakes by location and check the relationship between the occurrance of “earthquakes” and “tsunamis”. (As we know, some significant earthquakes can cause tsunamis.) We might be able to further identify the conditions needed for an earthquake to cause a tsunami.
