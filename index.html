Introduction to the Data

The dataset I’m using shows the GDPs of China and United States in US$ in recent 20 years. The data is downloaded from the World Bank database. The raw data file is uploaded to my GitHub page.

Data Preprocessing

  • Using the python pandas to read the csv file and drop the redundant columns
import pandas as pd
gdp = pd.read_csv('GDP.csv')
gdp = gdp.drop(['Series Name', 'Country Code'], axis=1)
  • Transpose the dataframe and reset the row index
gdp = gdp.transpose()
gdp = gdp.rename(columns=gdp.iloc[0]).drop(gdp.index[0])
gdp['year']=range(2001,2021)
GDP = gdp.set_index('year')
  • Display the GDP numbers in 10^10 $, and then save the processed data into “gdp_processed.csv” file to be uploaded to my GitHub page

Python code:

GDP['China'] = GDP['China']/10**10
GDP['United States'] = GDP['United States']/10**10
GDP.to_csv('gdp_processed.csv', index=True)
